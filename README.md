## Usage
1\. Choose a pre-trained model in *'trained_model'* folder.

2\. Modify paths for selected model and testing dataset in *'run.py'* file.

3\. Run **TBD**!!
```
python run.py
```

4\. If necessary, download pre-computed results at [here](https://drive.google.com/file/d/1tHe7YqYSo_F8W3ISP6pjh4RTG_bOg4sb/view?usp=sharing).
