## Usage
1\. Modify dataset paths in 'run.py' file.

2\. Select a model among the pre-trained models in 'trained_model' folder.

3\. Choose the testing datasets to validate the selected model. 

4\. Run TBD!
```
python run.py
```

5\. Download [pre-computed results](https://drive.google.com/file/d/1tHe7YqYSo_F8W3ISP6pjh4RTG_bOg4sb/view?usp=sharing) if necessary.
